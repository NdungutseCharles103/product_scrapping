# Product Web Scraper APIs

## Description

This is a web scraper API that scrapes data from a website and returns the data in JSON format. Curentlly it scrapes data from [https://www.ebay.com](https://www.ebay.com) and returns the data in JSON format.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
